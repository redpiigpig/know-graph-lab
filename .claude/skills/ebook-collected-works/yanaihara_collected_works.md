# 矢內原忠雄全集 — 案例盡職調查（版權 / 書目 / 來源）

神學學科開區案例之二（內村鑑三的弟子；師徒同開「神學」分區，2026-07-16）。**先讀本檔再開工。**

## 一句話結論

矢內原忠雄（Yanaihara Tadao, 1893–1961）是**內村鑑三門下無教會主義第二代領袖**＋日本殖民政策學頂尖學者＋戰後東京大學總長。卒於 1961 → **日本公有領域（舊著作權法卒後 50 年，2011 年底過期、2012 起 PD；2018 年 TPP 延長到 70 年不溯及既往）、台灣公有領域（卒後 50 年）**。⚠️ 美國因 URAA 回溯：1930 年後出版各卷受美國版權至出版後 95 年 → archive.org 無全文；但**《帝国主義下の台湾》1929 年出版 → 全球（含美國）公有領域**，且 NDL 已インターネット公開——**跟台灣直接相關的這部主著恰好是最乾淨可開工的一卷**。
