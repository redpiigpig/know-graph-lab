# 馬克思‧穆勒術語對照（英 / 德 / 繁中）

穆勒全集翻譯專屬 glossary。穆勒**以英文寫作為主**（牛津），故**英文原詞為準**，德文（穆勒監修《Einleitung》等）作對照；繁中採學界較通行譯名。翻譯前先鎖、翻完跑 term sweep 收斂變體（跨卷不可不一致）。

> 跟榮格 [jung_glossary.md](jung_glossary.md)（深層心理學）分開：穆勒是**比較宗教學／比較語言學／印度學**，自成體系。凡與 `/translation-glossary`（聖經人物/地名/神祇/吠陀文獻等）重疊者，**以 `/translation-glossary` `name_recommended` 為權威**（[[feedback_glossary_strict_authority]]）；本表只補穆勒專有的方法論術語。
