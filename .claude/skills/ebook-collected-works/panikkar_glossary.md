# 雷蒙‧潘尼卡術語對照（英 / 原文 / 繁中）

潘尼卡全集翻譯專屬 glossary。潘尼卡**用多種語言原創**（加泰隆／西／義／英／德／法），但本站採 **English-first**（[[feedback_jung_nonpd_english_first]]），故**英文為主對照詞**；繁中採學界較通行、或最貼合潘尼卡原意的譯名。翻譯前先鎖、翻完跑 term sweep 收斂變體（跨卷不可不一致；自鑄詞尤其）。

> 凡與 `/translation-glossary`（聖經人物/神祇/吠陀文獻/印度哲學名詞等）重疊者，**以 `/translation-glossary` `name_recommended` 為權威**（[[feedback_glossary_strict_authority]]）；本表只補潘尼卡專有的方法論與神學自鑄詞。與穆勒 [mueller_glossary.md](mueller_glossary.md)（比較宗教學）、榮格 [jung_glossary.md](jung_glossary.md)（深層心理學）分立。

## 招牌自鑄詞（最易亂，務必鎖死）

| 英文 | 原文／字源 | 繁中（採用）| 變體（收斂掉）| 備註 |
|---|---|---|---|---|
| cosmotheandric (intuition / vision / experience) | cosmos + theos + anēr/andros（宇宙‧神‧人）| 宇宙神人共融（cosmotheandric）| 宇宙神人論／神人宇宙／宇宙—神—人 | 潘尼卡核心直觀：實在恆為宇宙、神、人三維不可分的共融。**首見括注 (cosmotheandric)**，後文用「宇宙神人共融」 |
| theandric | theos + anēr（神‧人）| 神人（theandric）| | 基督論傳統詞，潘尼卡擴為 cosmotheandric |
| intrareligious dialogue | intra-（內）| 宗教內對話 | 宗教內在對話／教內對話 | **與 interreligious 嚴格區分**：在「我自己內部」與他者宗教相遇的對話（內在朝聖），非兩教派代表外部協商 |
| interreligious dialogue | inter-（間）| 宗教間對話 | 跨宗教對話 | 教派與教派之間的外部對話；**勿與 intrareligious 混譯** |
| Christophany | Christos + phainein（顯現）| 基督顯現（Christophany）| 基督顯／基督現象 | 潘尼卡自鑄，相對於 theophany（神顯）；指基督在萬有中的顯現，超越歷史耶穌的狹義基督論 |
| tempiternity | tempus + aeternitas（時間＋永恆）| 時永（tempiternity）| 時間永恆／永恆當下 | 潘尼卡：時間與永恆非對立，當下即永恆的綻露 |
| diatopical hermeneutics | dia + topos（跨地）| 跨地詮釋學（diatopical hermeneutics）| 異地詮釋學／越界詮釋 | 跨越不同文化「處所（topoi）」的詮釋方法，非同一傳統內的時間性詮釋 |
| homeomorphic equivalence | homeo + morphē（同形）| 同構等價（homeomorphic equivalence）| 同態對應／類比等值 | 潘尼卡比較方法：不同傳統中「功能位置相當」的概念對應（非字面翻譯對等），如 Brahman ↔ God 的功能類比 |
| ecosophy | oikos + sophia（家＋智慧）| 生態智慧（ecosophy）| 生態哲學 | 大地的智慧，非「關於生態的學問」 |
| pisteuma | pistis（信）類比 noumenon | 信識（pisteuma）| | 潘尼卡：信仰行為的對應客體（仿 noumenon/phenomenon 鑄詞）|
