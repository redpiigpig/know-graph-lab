## 本原與形上學（自然哲學 → 存有論）

| 希臘原文 | 音譯 | 繁中（採用）| 變體（收斂掉）| 備註 |
|---|---|---|---|---|
| ἀρχή | archē | 本原 | 始基／太初／原理 | 萬物的本原與起點（米利都學派核心問題）|
| ἄπειρον | apeiron | 無限者 | 無定／無限定／阿派朗 | 阿那克西曼德的本原；首字大寫指本原、小寫指「無限的」|
| φύσις | physis | 自然（本性）| 自然界 | 「論自然（Peri physeōs）」為前蘇格拉底通用書名 |
| λόγος | logos | 邏各斯 | 道／理／理法 | 赫拉克利特：宇宙的理法與言說；引《約翰福音》「道」時另註 |
| τὸ ἐόν / τὸ ὄν | to eon / to on | 存有（存在者）| 存在／有／是者 | 巴門尼德本體論核心；「存有」為採用譯 |
| οὐσία | ousia | 實體（本質）| 存在／本體／實有 | 亞里斯多德第一範疇；神學脈絡另有「本體」譯，翻時辨明 |
| εἶδος / ἰδέα | eidos / idea | 理型（相）| 理念／形式／型相 | 柏拉圖：採「理型」；亞里斯多德 eidos 作「形式」（與 hylē 質料對）|
| ὕλη | hylē | 質料 | 材料／物質 | 亞里斯多德四因之一 |
| δύναμις / ἐνέργεια | dynamis / energeia | 潛能／實現 | 潛勢／現實／隱德萊希 | 亞里斯多德；entelecheia 作「完全實現（隱德萊希）」|
| αἰτία | aitia | 因（原因）| 因由 | 四因說：質料因／形式因／動力因／目的因 |
| τέλος | telos | 目的（終點）| 目標 | 目的論（teleology）之本 |
| κίνησις | kinēsis | 運動（變化）| 動 | |
| τὸ Ἕν | to Hen | 太一 | 一者／一 | 普羅提諾新柏拉圖主義最高本體 |
| ἀπόρροια / πρόοδος | aporroia / proodos | 流溢 | 流出／發出 | 普羅提諾：太一流溢出智性、靈魂 |

## 知識、靈魂與心智

|---|---|---|---|---|
| νοῦς | nous | 努斯（心智）| 理智／心靈／心 | 阿那克薩哥拉宇宙心智；亞里斯多德／普羅提諾作「智性」時另註 |
| Νοῦς（本體）| Nous | 智性 | 理智 | 普羅提諾第二本體（含理型），首字大寫 |
| ψυχή | psychē | 靈魂 | 心靈／魂 | |
| λόγος（能力）| logos | 理性 | 理智 | 指人的理性能力時 |
| ἐπιστήμη / δόξα | epistēmē / doxa | 知識／意見 | 真知／臆見 | 巴門尼德「真理之路／意見之路」、柏拉圖知識論對立 |
| αἴσθησις | aisthēsis | 感覺（感官知覺）| 感性 | |
| ἀνάμνησις | anamnēsis | 回憶（說）| 記憶 | 柏拉圖《美諾》《斐多》：學習即回憶 |
| μετεμψύχωσις | metempsychōsis | 靈魂輪迴 | 輪迴轉世 | 畢達哥拉斯、恩培多克勒、柏拉圖 |
