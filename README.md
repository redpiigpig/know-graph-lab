# Know Graph Lab | 知識圖工作室

> Where Knowledge Takes Shape - AI-powered academic visualization tool

## 🎯 功能特色

- 📊 流程圖編輯器
- 🧠 心智圖工具
- 🗺️ 地理地圖標記（AI 智能識別）
- 🌳 譜系圖繪製
- 📊 智能表格
- 🎨 AI 生成簡報並導出 PPT

## 🚀 快速開始

\\\ash
# 安裝依賴
npm install

# 設定環境變數
cp .env.example .env

# 啟動開發伺服器
npm run dev
\\\

## 🛠️ 技術棧

- **前端**: Nuxt 3 + Vue 3 + TypeScript
- **後端**: Nitro (Nuxt 內建)
- **資料庫**: Supabase
- **AI**: Anthropic Claude API
- **地圖**: Leaflet.js + GADM/Natural Earth 資料
- **部署**: Zeabur
